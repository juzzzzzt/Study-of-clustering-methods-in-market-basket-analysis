# Study-of-clustering-methods-in-market-basket-analysis
Term Paper by Ivan Churilov

📁 Repository Description
This repository contains data files and the main codebase for processing intermediate results and generating final summary tables as part of a data analysis project.

📂 Project Structure
.xlsx files — Excel files containing intermediate calculation results and final output tables used for reporting or visualization.
project.ipynb — The core Jupyter Notebook of the project. It includes the full logic for data loading, transformation, analysis, and result generation.
💡 Purpose
The purpose of this repository is to maintain all versions of the data throughout the processing pipeline and ensure full reproducibility of the analytical workflow through a single notebook file.
